# 포트폴리오 사이트입니다.
http://sungmin123.loca.lt ( localtunnel )


#### 1. 로그인, 회원가입
로그인 기능과 회원가입 기능을 구현했습니다.
  
#### 2. 게시판
게시판 기능을 구현했습니다.
 
#### 3. 영화boxOffice
영화의 순위와 영화검색 기능을 구현했습니다.

#### 4. imgSound
랜덤 이미지 와 오디오의 파형을 보여주는 기능을 구현했습니다.

  외부 api를 사용하여 랜덤 이미지를 가져옵니다.
  캔버스를 사용하여 이미지에서 가장 많은 색을 추출합니다.
  오디오 관련 함수를 사용하여 음악의 파형을 구현합니다.
